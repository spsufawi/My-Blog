10年高考数学结束后自认为考的不错，最后一道大题做出了两问，整体不难，所以自认为能有140+, 结果出来只有128，后来查了下答案，填空题第一题就错了，后边一道线性规划答题也没算对。线性规划的题先不说，来看下这道填空题。

我似乎找到我大学cs学不好的原因了，稍微观察下就知道输出的S实际就是\(x_1,x_2,x_3,x_4\)平均值，我们用R模拟下。

s <- 0
i <- 1
x <- c(1, 1.5, 1.5, 2)
while(i <= 4){
  s <- s + x[i]
  S <- s / i
  i <- i + 1
}
print(S)
## [1] 1.5
